# salva-el-mundo-
# 🌀 Proyecto Fusión: STW + TSB + JJS

Este repositorio contiene las mecánicas base para un juego que fusiona la construcción de *Fortnite: Salvar al Mundo* con el combate avanzado de *The Strongest Battlegrounds* y *Jujutsu Shenanigans*.

## 📁 Estructura del Proyecto
- `BuildingSystem.lua`: Sistema de cuadrícula y edición de estructuras.
- `CombatSystem.lua`: Combos M1, Ragdoll y detección de habilidades.
- `SkillBuilder.lua`: Menú de personalización de Hotbar y modificadores pasivos.
